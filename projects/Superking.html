Super King - A Fan-Based Website

An ASP.NET Core Project

Description

A fun little website I made dedicated to one of my favorite super markets. I created this webpage because I wanted to learn how to:

  • Create an ASP.NET Core MVC Application
  • Utilize the Newtonsoft.Json package

For this project, I recommend reading the READ.ME file that is located on the repository.

I used the Newtonsoft.Json package learn how to serialize and unserialize data.

All in all, it was a fun project I did to get more comfortable with ASP.NET Core. This application helped better my understanding of how the MVC model works, and how each part communicates with one another. If you or if anyone else has any other recommendations on how I can better my website, please let me know.
